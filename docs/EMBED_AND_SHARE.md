### WordPress Shortcode (Custom)

If you control your WordPress site, add this to `functions.php`:

```php
function tire_calculator_embed($atts) {
    $a = shortcode_atts(array(
        'current' => '265/70R17',
        'new' => '285/75R17',
        'gear' => '',
        'width' => '100%',
        'height' => '400'
    ), $atts);

    $url = 'https://overlandn.com/tires/embed.html?current=' . urlencode($a['current']) . '&new=' . urlencode($a['new']);
    if (!empty($a['gear'])) {
        $url .= '&gear=' . urlencode($a['gear']);
    }
    $url .= '&auto=true';

    return '<iframe src="' . esc_url($url) . '" width="' . esc_attr($a['width']) . '" height="' . esc_attr($a['height']) . '" frameborder="0" style="border: 1px solid #333; border-radius: 8px;"></iframe>';
}
add_shortcode('tire_calc', 'tire_calculator_embed');
```

**Usage in posts:**
```
[tire_calc current="265/70R17" new="285/75R17" gear="3.73"]
```
